# ☕ Bright Coffee Shop Analytics

## 📌 Project Overview

This project is an end-to-end data analysis of 6 months of coffee shop transaction data from New York City. The goal was to uncover sales trends, customer behaviour, and business performance insights.

## 🎯 Objectives

* Analyze sales performance over time
* Identify peak hours and high-performing products
* Understand customer purchasing patterns
* Provide actionable business insights

## 🛠 Tools & Technologies

* SQL (Databricks)
* Excel
* Data Cleaning & Transformation
* Data Visualization

## 🔍 Process

1. Cleaned and prepared raw transactional data
2. Performed SQL analysis to explore trends
3. Aggregated data for reporting
4. Created visual insights for decision-making

## 📊 Key Insights

* Peak sales occur during morning hours
* Certain products consistently outperform others
* Weekends show higher revenue patterns

## 📈 Outcome

This analysis can help optimize staffing, inventory planning, and marketing strategies.
